# React, Node, MongoDB application which:
1) Authenticates Users
2) Creates & Gets (Questions & Answers) Unit

# Install
1) Install MongoDB
2) cd backend => npm i
3) cd frontend => npm i

# Run & Usage
1) cd backend => node src
2) cd frontend => npm start
